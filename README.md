# Jiradash
Jira task dashboard by dash.

## Installation

```
pip install -r requirements.txt
```

## Usage

Setting environment.

```
$ export JIRA_URL="https://projectName.atlassian.net"
$ export JIRA_PJ="ProjectName"
$ export JIRA_USER="user"
$ export JIRA_PASS="password"
```

Run application.

```
$ python app.py
```
